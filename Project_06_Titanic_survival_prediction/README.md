# Titanic Survival Prediction

This project uses Machine Learning (Random Forest) to predict whether a passenger survived the Titanic disaster based on their demographic and ticket information.

---

## 📘 Project Overview

The goal is to predict survival (0 = Died, 1 = Survived) using features like:

- Passenger Class (1st, 2nd, 3rd)
- Sex
- Age
- Number of siblings/spouses aboard
- Number of parents/children aboard
- Ticket fare
- Port of embarkation (C, Q, S)

The project includes:

- Data cleaning (handling missing values)
- Feature encoding (converting text to numbers)
- Model training using Random Forest
- An interactive Streamlit web app

---

## 🧠 Technologies Used

- Python 🐍
- pandas, NumPy
- scikit-learn (RandomForest, LogisticRegression)
- matplotlib, seaborn
- joblib
- streamlit

---

## 📂 Dataset

**Titanic: Machine Learning from Disaster** (Kaggle)

The dataset contains 891 training samples with 12 features.

🔗 **Source:** [Kaggle Titanic Dataset](https://www.kaggle.com/c/titanic)
